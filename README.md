# SecureChat 本地账号注册与登录示例

这个项目已重写为本地账号认证系统演示，重点实现：

- 本地 SQLite 数据库存储账号信息
- 注册时填写真实姓名、手机号、身份证号
- 注册时生成本地6位验证码完成验证
- 登录时可使用密码或验证码登录
- 每次注册和登录都将账号信息写入本地数据库

---

## 运行方式

```bash
pip install -r requirements.txt
python main.py
```

---

## 功能说明

- 注册页面要求填写手机号、密码、真实姓名和身份证号。
- 点击“发送验证码”后会生成本地验证码，并通过弹窗提示用户输入。
- 登录时可选择使用密码登录，或使用验证码登录。
- 数据库文件 `auth.db` 会保存在项目根目录下。

---

## 目录结构

```
securechat/

```

## 注意

- 验证码功能为本地模拟：验证码在程序内生成并显示给用户，未集成短信网关。
